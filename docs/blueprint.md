# **App Name**: InteriorGenius

## Core Features:

- Personalized Consultation Booking: Allows users to book a free consultation by providing their name, email, and phone number.
- AI Interior Design Tool: The application incorporates an AI-powered design tool. This feature allows users to upload a photo of a room, describe their aesthetic preferences, and generate interior design ideas, furniture suggestions, and layout options to guide interior decorating using a tool
- Dynamic Kitchen Quote: Calculates a free kitchen quote based on layout options and selected features.
- Room Visualizer: Helps customers view living rooms, bedrooms, and more to help select appropriate setups and modifications to each location
- FAQ Section: Provides quick answers to common questions about project timelines, small home solutions, and maintenance.

## Style Guidelines:

- Primary accent: Gold/Yellow (#B89C4E) for CTAs, highlights, borders, and icons.
- Secondary accent: Green Dot (#608D38) used sparingly for trust marks, checkmarks, and status indicators.
- Background: Dark Charcoal/Black (#1A1A1A) for the main page background, providing a rich and dramatic feel.
- Text/Content: White (#FFFFFF) for headings and body text to ensure maximum readability.
- Headline font: 'Playfair', serif, for elegance; body font: 'PT Sans', sans-serif, for readability.
- Use minimalist, line-style icons to represent various services and features, with Green Dot accent for approval.
- Clean and spacious layout with clear visual hierarchy, focusing on readability and ease of navigation. Dark charcoal backgrounds with white text.
- Subtle transitions and animations for a smooth and engaging user experience.
